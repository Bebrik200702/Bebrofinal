Запуск решения:
1) Для начала вам нужно установить нужные либы и скачать датасет, для этого запустите ноутбук load_data.ipynb
2) Далее вам нужно почистить датасет, удалив из него семплы в которых <= 3 точек сегментации, для этого запустите ноутбук ClearValidation.ipynb
3) Далее вам нужно преобразовать датасет в формат YOLO, для этого запустите ноутбук BuildYOLORegim.ipynb
4) Далее вам нужно разбить датасет на трейн и вал, для этого запустите ноутбук create_new_data.ipynb
5) Дале вам нужно обучить модель YOLO - это мы можете сделать в ноутбуке Train_YOLO_Full.ipynb
6) Далее мы можете сделать инференс обученной модели, поменяв веса и запустив ноутбук YOLO_TTA_Infer.ipynb
